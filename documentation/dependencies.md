### Dependencies

```
- Amazon IAM credentials must be created to enable access to the Elastic Beanstalk CLI: Give the user Admin access

- Node v14.15.1 (LTS) or more recent. While older versions can work it is advisable to keep node to latest LTS version

- npm 6.14.8 (LTS) or more recent, Yarn can work but was not tested for this project

- AWS CLI v2

- A RDS database running Postgres with inbound security rules set to allow 0.0.0.0/0 for all traffic.

- A S3 bucket for hosting uploaded pictures, ensure bucket is public.

```
